# Credit Card Fraud Detection Dashboard

## Overview

This project presents an interactive Power BI dashboard designed to analyze credit card transactions and detect fraudulent activities. The dashboard delivers meaningful business insights through dynamic visualizations, key performance indicators (KPIs), and interactive filters, enabling efficient monitoring of transaction behavior and fraud patterns.

## Objectives

- Analyze credit card transaction data effectively.
- Identify fraudulent transaction patterns.
- Monitor transaction trends and fraud distribution.
- Deliver actionable insights through interactive visualizations.

## Tools & Technologies

- Microsoft Power BI
- Power Query
- DAX (Data Analysis Expressions)

## Key Features

- Interactive KPI Cards displaying critical business metrics.
- Daily Transaction Trend Analysis.
- Fraud vs Genuine Transaction Comparison.
- Fraud Distribution by Entry Mode.
- Merchant Category Analysis.
- Transaction Amount Analysis.
- Dynamic slicers for Date, Entry Mode, Fraud Status, and Network ID.

## Business Insights

- Identifies high-risk transaction categories.
- Highlights fraud distribution across different payment entry modes.
- Tracks transaction trends over time.
- Supports data-driven decision-making through interactive reporting.
